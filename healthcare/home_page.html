 MEDICAL DONATION

This module deals with the medical related donation.
It checks the eligibility of user for donating blood, plasma cells, etc.
Then it guides the user to the nearby donation centre. Then the user selects the donation
centre for donating and all the
information of this centre can be downloaded by the user from this module.

This module also deals with the medical diagnosis.
